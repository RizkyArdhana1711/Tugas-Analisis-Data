Persyaratan Portofolio Penelitian Laboratorium Lengkap
Menyelesaikan Persyaratan Portofolio Penelitian Laboratorium

Tujuan

Dalam lab ini, Anda akan menjelajahi Portofolio Analis Data dan membuat daftar periksa untuk menunjukkan keterampilan analis data.

Bagian 1: Portofolio Analis Data Riset

Bagian 2: Membuat Daftar Periksa Keterampilan Analis Data

Latar Belakang / Skenario

Portofolio analisis data menyoroti pekerjaan Anda dan menampilkan personal branding serta keterampilan komunikasi Anda. Seorang analis data juga perlu memiliki rencana untuk mengembangkan keterampilan yang diperlukan dan untuk melaksanakan serta mempresentasikan proyek mereka kepada calon pemberi kerja. Gunakan lab ini sebagai titik awal untuk memulai Portofolio Analis Data Anda. Seiring kemajuan Anda dalam kursus ini, Anda dapat menggunakan aktivitas di dalam kursus untuk membangun proyek analitik pertama Anda.

Sumber Daya yang Dibutuhkan

  • Perangkat seluler atau PC/laptop dengan browser dan koneksi internet.

Instruksi

Bagian 1: Portofolio Analis Data Riset

Langkah 1: Cari portofolio analis data.

  1. Dengan menggunakan peramban web, cari contoh portofolio ilmu data .
  2. Pilih beberapa portofolio dan bacalah serta perhatikan beragam konten dan gaya presentasinya.
    Situs web https://careerfoundry.com/en/blog/data-analytics/data-analytics-portfolio-examples/ membahas contoh-contoh portofolio.

Langkah 2: Buat daftar proyek yang mungkin.

Buatlah daftar beberapa proyek menarik yang disorot dalam portofolio yang telah Anda lihat.

Sertakan juga proyek-proyek yang mungkin ingin Anda kerjakan, proyek-proyek di mana Anda ingin melakukan riset data, seperti hobi, olahraga, atau minat lainnya.

Langkah 3: Pertimbangkan kemungkinan gaya portofolio.

Pertimbangkan bagaimana Anda ingin mempresentasikan proyek-proyek Anda dalam portofolio. Pilih gaya yang mencerminkan kepribadian dan tujuan Anda.

Periksalah contoh-contoh yang telah Anda temukan untuk menentukan gaya media dan tata letak yang mewakili diri Anda sesuai dengan harapan calon pemberi kerja.

Langkah 4: Buat daftar periksa portofolio.

  1. Buatlah daftar periksa komponen-komponen yang Anda lihat dalam portofolio yang telah Anda tinjau.
  2. Saat Anda mengerjakan aktivitas dalam kursus ini, perbarui daftar periksa setiap kali Anda menyelesaikan setiap item dalam daftar tersebut.

Bagian 2: Membuat Daftar Periksa Keterampilan Analis Data

Langkah 1: Cari keterampilan pekerjaan analis data.

Cari lowongan pekerjaan analis data di negara atau wilayah Anda melalui internet.

Salah satu situs yang dapat dipertimbangkan adalah https://www.ziprecruiter.com/n/Data-Analyst-Jobs-Near-Me .

Situs seperti https://www.thinkful.com/blog/entry-level-data-analyst/ juga bermanfaat.

Perhatikan persyaratan pekerjaan dan keterampilan penting untuk analis data tingkat pemula atau junior. Beberapa keterampilan yang perlu dicari meliputi:

  • SQL

  • Microsoft Excel

  • Statistik

  • Visualisasi data

  • Keterampilan presentasi dan komunikasi

  • Berpikir kritis

Penting untuk membedakan bagaimana keterampilan ini berkaitan dengan peran tingkat pemula dan harapan untuk peran yang lebih berpengalaman.

Langkah 2: Buat daftar periksa keterampilan analis data.

Buatlah daftar periksa persyaratan pekerjaan untuk peran analis data. Saat Anda mempelajari dan mempraktikkan setiap keterampilan, catat tingkat kemahiran Anda – mungkin gunakan skala seperti tidak mahir, dasar, menengah, dan mahir.

Perbarui terus resume dan portofolio Anda seiring kemajuan Anda dalam daftar periksa.

    Bagian 3: Menyiapkan Repositori GitHub untuk Portofolio Anda

    GitHub adalah platform hosting yang menyediakan layanan kolaborasi dan kontrol versi untuk individu dan organisasi. Pada bagian ini, Anda akan membuat akun GitHub pribadi dan repositori untuk penggunaan Anda.

    Langkah 1: Jelajahi GitHub.

    1. Buka dokumentasi GitHub atau cari dokumentasi GitHub menggunakan peramban web.
    2. Klik Mulai untuk meninjau beberapa fitur dasar GitHub.
      Jenis akun (produk) apa saja yang tersedia di GitHub?
      Area Jawaban
      There are three types of accounts on GitHub. Personal accounts, Organization accounts, and Enterprise accounts.
      Apa yang bisa Anda lakukan dengan akun pribadi?
      Area Jawaban
      Your personal account can own resources such as repositories, packages, and projects. Any time you take any action on GitHub.com, such as creating an issue or reviewing a pull request, the action is attributed to your personal account.

      Langkah 2: Buat akun pribadi.

      Akun GitHub diperlukan untuk membuat repositori. Repositori adalah tempat Anda dapat mengatur proyek dan menyimpan file serta folder untuk satu proyek.

      Pada langkah ini, Anda akan membuat akun GitHub pribadi.

      1. Kunjungi GitHub untuk mendaftar akun.
        Catatan : Jika Anda sudah memiliki akun GitHub, pastikan Anda sudah keluar dari akun tersebut.
      2. Pada kolom Alamat email , masukkan alamat email yang valid dan klik Daftar untuk GitHub untuk melanjutkan ke layar Selamat Datang.
      3. Di layar Selamat Datang, verifikasi alamat email Anda, buat kata sandi, masukkan nama pengguna, jawab pertanyaan tentang menerima produk dan pembaruan melalui email, dan selesaikan teka-teki. Klik Buat akun untuk membuat akun pengguna baru. Anda akan menerima kode dari GitHub di alamat email yang Anda berikan.
      4. Masukkan kode peluncuran yang diberikan dan akun Anda siap digunakan.
      5. Jawab pertanyaan pribadi dan klik Lanjutkan . Klik Lanjutkan pada layar fitur tanpa memilih apa pun. Klik Lanjutkan gratis setelah meninjau fitur akun gratis.

      Langkah 3: Buat repositori pribadi baru.

      Sekarang setelah Anda memiliki akun pengguna baru, Anda akan membuat repositori pribadi baru.

      1. Masuk ke GitHub menggunakan akun pengguna baru Anda jika diperlukan.

      2. Klik Buat repositori baru .

      3. Masukkan nama repositori pilihan Anda di kolom Nama repositori . Berikan deskripsi repositori jika diinginkan. Pilih Pribadi untuk membuat repositori pribadi. Pilih Tambahkan file README . Klik Buat repositori untuk melanjutkan.

      Langkah 4: Kelola file di repositori.

      File dapat diunggah ke repositori atau membuat file baru di repositori. Jika Anda sudah membuat file README, Anda dapat menyalin isi file README tersebut ke dalam file README yang sudah ada di repositori.

      Pada langkah ini, Anda akan mengunggah file ke repositori. Untuk informasi lebih lanjut tentang manajemen file, lihat dokumentasi GitHub mengenai repositori. ( https://docs.github.com/en/repositories ).

      1. Di repositori yang baru dibuat, klik Tambah file > Unggah file untuk mengunggah file baru di bawah tab Kode.

      2. Anda dapat menyeret atau mengklik file yang ingin Anda tambahkan ke repositori. Klik Commit changes untuk menyimpan perubahan langsung ke cabang utama.

      Pertanyaan Refleksi

      Renungkan keterampilan yang dibutuhkan oleh peran analis data dan pertimbangkan keterampilan tambahan, atau yang lebih canggih, yang diperlukan untuk mengembangkan proyek-proyek untuk portofolio Anda yang menarik minat Anda dan ke mana Anda melihat karier analis data Anda akan membawa Anda.

      Aktivitas Tantangan

      Proyek teleskop Square Kilometer Array (SKA) ( https://www.skatelescope.org/the-ska-project/ ) dan Teleskop James Webb ( https://www.nasa.gov/mission_pages/webb/main/index.html ) adalah proyek astronomi jangka panjang yang akan menghasilkan volume data yang hampir tak terbayangkan selama masa berjalannya masing-masing proyek.

      Pikirkan tentang keterampilan data yang perlu diterapkan untuk menganalisis data ini, dan proses apa yang mungkin perlu dikembangkan untuk mengelola analisis ini di masa mendatang.
